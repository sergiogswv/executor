## 📋 Descripción

El Executor es el agente encargado de transformar las decisiones de Cerebro en código real. Utiliza una integración profunda con **Aider** y herramientas de terminal para aplicar parches, resolver bugs y realizar refactorizaciones autónomas.

### Funcionalidades Clave:
- **Auto-Fix Integration:** Conexión nativa con Aider para aplicaciones de código con LLM.
- **Real-time Logs:** Streaming de la salida de terminal directamente al Dashboard de Skrymir.
- **Safe Mode:** Creación de backups automáticos antes de cada intervención.
- **Git Flow:** Gestión automática de ramas para cada feature o fix implementado.
